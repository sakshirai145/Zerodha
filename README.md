# Zerodha Clone (MERN Stack)

A full-stack stock trading dashboard inspired by Zerodha.

## Tech Stack

- React.js
- Node.js
- Express.js
- MongoDB (Atlas)
- Axios
- Mongoose

## Features

- View Holdings
- View Positions
- Place Buy Orders
- REST API Backend
- MongoDB Database Integration
- Buy/Sell Order Management

## Project Structure

Zerodha/
│
├── backend/       → Express + MongoDB API
├── dashboard/     → Trading dashboard (React)
├── frontend/      → Landing page (React)
│
└── README.md

## Installation & Setup

### 1️⃣ Clone the repository
git clone https://github.com/your-username/Zerodha.git

### 2️⃣ Backend Setup
cd backend
npm install
npm start

Runs on: http://localhost:3002

### 3️⃣ Dashboard Setup
cd dashboard
npm install
npm start

Runs on: http://localhost:3000

### 4️⃣ Landing Page Setup
cd frontend
npm install
npm start
